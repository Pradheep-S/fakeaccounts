Expected Data Format

CSV File Requirements:

Your CSV file should include the following columns (case-insensitive):

Required fields:
  • username - Account username/handle
  • created_at - Account creation date (YYYY-MM-DD format)
Optional fields (enhance detection accuracy):
  • email - Account email address
  • full_name - User's full name
  • bio - Profile biography
  • followers - Number of followers
  • following - Number of accounts following
  • posts - Total number of posts
  • profile_picture - Profile picture URL
  • email_verified - Email verification status (true/false)
  • phone_verified - Phone verification status (true/false)
  • website - User's website
  • location - User's location

JSON File Requirements:

Your JSON file should be an array of account objects:

[
  {
    "username": "john_doe123",
    "email": "john@example.com",
    "full_name": "John Doe",
    "bio": "Software developer",
    "followers": 150,
    "following": 200,
    "posts": 85,
    "created_at": "2023-05-15",
    "email_verified": true,
    "phone_verified": false,
    "profile_picture": "https://example.com/profile.jpg"
  }
]
